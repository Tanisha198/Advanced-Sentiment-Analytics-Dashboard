# AI-Sentiment-Analyzer
Sentiment Analysis Web App built with Python, NLP, and Streamlit. Users can enter multiple comments or upload a text file, and the app predicts sentiment (Positive / Negative / Neutral) with a visual distribution chart.

A simple and interactive Sentiment Analysis Web App built using Python, NLP, Streamlit, and Matplotlib.  
This app allows users to analyze the sentiment of multiple comments — either entered manually or uploaded through a text file.

It predicts whether a comment is Positive, Negative, or Neutral, and displays a visual sentiment distribution chart.

## Features

-  Real-time sentiment detection
-  Streamlit-based clean UI
-  Multiple comment input support
-  Visual sentiment distribution chart using Matplotlib
-  Python + NLP workflow
-  Fast, lightweight, and beginner-friendly UI built with Streamlit

## Technologies Used

- Python | Core programming language 
- Streamlit | Web app development 
- NLP | Sentiment analysis 
- Matplotlib | plotting sentiment distribution chart 
- File Handling | Reading uploaded text files
